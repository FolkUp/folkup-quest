# Phase 0.5 Quality Scoring Rubric

**Reviewers:** Андрей (client approval) + Art Director (technical assessment)  
**Scale:** 1-10 per dimension, weighted average for final score  
**Threshold:** 6.0+ for AI-first approval, 5.0-5.9 for hybrid consideration, <5.0 for professional approach

## Scoring Dimensions (Weighted)

### 1. Sin City Style Adherence (25% weight)
**10:** Perfect B&W high contrast, spotted blacks, negative space mastery  
**8:** Strong B&W contrast, minor gray intrusion (<5%)  
**6:** Recognizable Sin City style, moderate gray areas (5-15%)  
**4:** Some noir elements but gray dominance (15-30%)  
**2:** Minimal style adherence, mostly gray illustration  
**1:** No Sin City style recognition

### 2. Character Accuracy (25% weight)
**10:** Perfect match to Character Bible description (Арни: thin/wiry/ponytail/leather jacket)  
**8:** Minor deviations in secondary details (jacket texture, beard length)  
**6:** Core identity intact, 1-2 major details incorrect  
**4:** Character recognizable but significant appearance drift  
**2:** Character barely recognizable from description  
**1:** Character completely inaccurate

### 3. Composition/Camera Work (20% weight) 
**10:** Exact storyboard camera angle, perfect panel framing  
**8:** Correct angle with minor framing adjustments  
**6:** Similar angle, composition serves story purpose  
**4:** Functional composition but deviates from storyboard intent  
**2:** Poor composition undermines storytelling  
**1:** Compositionally broken, unreadable

### 4. Technical Quality (20% weight)
**10:** Clean lines, perfect resolution, no artifacts  
**8:** High quality with minor technical flaws  
**6:** Good quality, some artifacts but publishable  
**4:** Functional but noticeable technical issues  
**2:** Poor technical execution, major artifacts  
**1:** Unpublishable technical quality

### 5. Emotional Impact (10% weight)
**10:** Perfect mood capture from storyboard description  
**8:** Strong emotional reading, minor mood deviation  
**6:** Adequate mood representation  
**4:** Weak emotional impact  
**2:** Wrong emotional tone  
**1:** No emotional coherence

## Scoring Protocol

1. **Independent review:** Both reviewers score separately
2. **Variance check:** If scores differ by >2 points on any dimension, discuss and reconcile
3. **Final calculation:** Weighted average: (Style×0.25) + (Character×0.25) + (Composition×0.20) + (Technical×0.20) + (Emotion×0.10)
4. **Production decision:**
   - **6.0-10.0:** AI-first approach approved ($20-100 budget)
   - **5.0-5.9:** Hybrid approach recommended ($2.8K-5.1K budget) 
   - **<5.0:** Professional approach required ($4K-9K budget)

## Character Consistency Protocol

**Calibration panel:** Generate Panel 8.1 (Alice Reveal) first - establishes character baseline  
**Validation panels:** Include Panel 2.3 (Арни at bar) for consistency check across same character  
**Consistency scoring:** Deduct 1 point from Character dimension if same character varies >20% between panels
