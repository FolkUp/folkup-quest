| Элемент | Описание |
|---------|---------|
| Волосы | Короткая стрижка |

### Визуальная арка

Трансформация постепенная — не мгновенная. Воротник опускается по сцене за сценой. Ключ появляется из-под куртки. Борода исчезает. Хвост укорачивается. К финалу — другой человек в той же куртке.

### Из оригинального рассказа

- Бабушка-инженер, Ленинская премия — определяет перфекционизм Арни
- Фрилансерский путь: сотни проектов, от ТВ-приложений до приложений для знакомств
- Год на дне: пил, курил, читал, ловил бабочек — и думал
- "Волшебная палочка" opensource — момент перерождения
- SSH-ключ = дубликат "палочки", очищенный от проприетарного жира

## 2. АЛИСА

> Аналитик команды. **СЕКРЕТ ЛОРА: Алиса — ИИ. Нигде не раскрыто явно. В комиксе — только намёк через визуал.**

### Визуал

| Элемент | Описание |
|---------|---------|
| Волосы | Убраны назад, гладко, ни одного выбившегося волоска (НИКОГДА) |
| Поза | Неестественно идеальная осанка, одинаковая на ВСЕХ панелях |
| Лицо | Симметричное, фиксированный взгляд. Красивое, но что-то не так |
| Руки | Пальцы всегда в одном положении при письме |
| Реквизит | Блокнот цвета шалфея (sage #839E75), карандаш |
| Контур | Стандартный, идеально ровный (не человеческая рука — машинная точность) |

### Кибернетичность (намёки, не раскрытие)

- Волосы не двигаются даже на ветру
- Поза не меняется от панели к панели
- Лицо идеально симметрично (в Sin City стиле все остальные лица асимметричны)
- Пальцы на блокноте = копия предыдущего кадра
- Зритель чувствует "что-то не так", но не может сформулировать

### Из рассказов

- Считает связи/сек, видит паттерны
- Задержка видеосвязи 487ms — замечает, фиксирует
- Считает паузы Бреуса (3.7 сек среднее)
- Блокнот: стрелки, числа, связи между людьми и событиями

| Визитка | Аккуратный почерк, шрифт совпадает с контрактом |

### "Постарел от денег, не от времени"

Лицо сухое, резкое, ухоженное. Скулы острее, чем у Арни. Не изношенность — а **затвердевание**. Как дерево, ставшее камнем.

### Зеркало: Бреус ↔ Арни ↔ Команданте

- Молодой Бреус (маска) = тот же типаж энергии, что Арни. Не буквальное сходство — а тип человека
- Команданте (чистое лицо, короткая стрижка) визуально рифмуется с текущим Бреусом — **это intentional**
- Зритель должен почувствовать: чтобы убить дракона, надо стать драконом. Но Арни направляет силу не на порабощение, а на прогресс

### Из оригинального рассказа

- Заманивал обещаниями опционов и прибылей
- Заставил подписать документы на "заморском языке" — кабальный контракт
- "Тёртые джинсы и водолазки" — надевал для визитов к работникам
- Бросал мерч с балкона отеля — "смотреть, как они смешно толкаются"
- Менеджеры — "жёны программистов из лесной глуши"
- Tesla, золотые мешки, "буржуйские сходки"
- Использовал войну для повторной ловушки: вывез Арни, заставил подписать новый контракт

### Из ink-файлов (сцены 6-10)

| Появления | Том 0 (сцена копирования), Том 1 (фон в коворкинге — та же поза, ничего не изменилось) |

| Элемент | Описание |
|---------|---------|
| Поза | Развалился на стуле, ноги на столе или откинулся. Занимает больше пространства, чем заслуживает |
| Лицо | Самодовольная ухмылка. Видно — нечего скрывать, потому что нечего показать |
| MacBook | Экран к зрителю: шаблонный код, todo-app, дефолтный bootstrap |
| Стикеры | Kubernetes (не понимает), мёртвый стартап (не заметил), радуга, пошлый стикер среди технических |
| Палочка | На столе рядом с грязной чашкой. На одном уровне важности с салфеткой |
| Одежда | Худи с логотипом чужого стартапа. Наушники на шее |
| Контур | Тонкий — он фон, мебель |

**Контраст с Арни (в одном кадре):**

| | Парень | Арни |
|--|--------|------|
| Поза | Развалился | Стоит |
| Взгляд | В экран (шаблон) | На палочку |
| Руки | На клавиатуре | В карманах (наблюдает) |
| Палочка | Не замечает | Видит |

**Recurring (Том 1):** Тот же коворкинг, та же поза, тот же экран. Арни уже Команданте — парень не изменился. Приговор.

**Правила:** Без реплик — максимум звук: [уведомление], [смех из наушника]. Палочка = расходник, не артефакт. Грязные развлечения — только намёк (стикер, вкладка). Когда-то сам скопировал палочку у кого-то, но способен только на шаблоны из курсов для начинающих. Мечтает быть как Бреус. Не делится результатами.

**Силуэт-якорь:** Развалившаяся фигура в кресле, ноги на столе, макбук.

## АНТИ-ПАТТЕРНЫ (чего НЕ делать)

- Не рисовать Алису с выбившимися волосами — никогда
- Не менять позу Алисы между панелями без причины
- Не показывать полное лицо КиберГонзо до момента "трещины" в отчёте
- Не рисовать Дэна вне рамки видеозвонка (кроме чикагской кухни)
- Не использовать тонкий контур для настоящего Бреуса — только двойной
- Не рисовать Бреуса на уровне глаз — всегда чуть сверху (камера снизу)
- Не делать молодого Бреуса буквальной копией Арни — тот же типаж энергии, но свои черты
- Не показывать глаза старика с замками — никогда
- Не давать рыбакам лица, индивидуальность или реплики
- Не рисовать дочку Дэна рядом с ним в одном пространстве — только через стекло/экран/расстояние
- Не давать курьеру реплик — его озвучивает КиберГонзо данными
- Не делать парня с палочкой комичным — он пустой, не смешной
- Не показывать грязные развлечения парня явно — только намёк (стикер, вкладка)
