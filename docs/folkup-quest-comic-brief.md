> **Дата:** 01.03.2026

## Концепция

**Профессиональный нуар-комикс в стиле Sin City** по мотивам FolkUp Quest.
Не пересказ игры, а **расширение вселенной** — как Witcher-комиксы расширяют игру, не дублируя.

**Формат:** 20–30 страниц, чёрно-белый с цветовыми акцентами (bordeaux/sage/amber — палитра игры).

---

## Стиль: Sin City — что именно

### Техническая ДНК Фрэнка Миллера

| Приём | Как работает | Пример |
|-------|-------------|--------|
| **Чистый ч/б** | Нет полутонов, нет серого. Только чёрное и белое. Глаз достраивает остальное | Сцена Марва на мосту — видны контуры здания, хотя они не нарисованы |
| **Негативное пространство** | Рисуется то, чего НЕТ. Фигура = белая пустота на чёрном фоне | Силуэты между стенами из кирпича |
| **Spot color** | Один цвет на весь том — акцентирует персонажа или тему | Красный = кровь (Hard Goodbye), Жёлтый = злодей (That Yellow Bastard) |
| **Звук = архитектура** | "BLAM" не написано рядом — буквы ФОРМИРУЮТ контуры панелей | Выстрел в церкви — буквы "BLAM" образуют силуэт креста |
| **Зубная щётка** | Тушь на щётку → палец → брызги = текстура дождя, стен, неба | Каждая страница уникальна — controlled chaos |
| **Двойной контур** | Линия контура удваивается → ощущение тяжести и грубости | Руки Марва, лица злодеев |

### Что отличает Sin City от "просто нуара"

- **Нет серого** — ни визуально, ни морально
- **Минимум фонов** — пространство создаётся тенями и силуэтами
- **Манга-гибрид** — Миллер считал американские комиксы "слишком многословными", японские "слишком пустыми", создал гибрид
- **Текст-боксы = внутренний монолог** — не диалог, а hardboiled narration
- **Ломаная сетка панелей** — splash-страницы для кульминаций, плотная сетка для напряжения

|--------|-------|-----------|
| **100 Bullets** | Azzarello/Risso | Chiaroscuro, узнаваемые силуэты персонажей при большом касте |

## Применение к FolkUp Quest

### Spot color — три цвета = три пути

| Концовка | Цвет | Что акцентирует |
|----------|------|-----------------|
| Фонарь (folk) | **Sage #839E75** | Лампа фонаря, блокнот Алисы, зелёные глаза |
| Мост (neutral) | **Amber #E8AD4A** | Гитарные струны Дэна, огонь сигареты, уличный свет |
| Кресло (dragon) | **Bordeaux #7D4450** | Визитка Бреуса, кресло, красные цифры контракта |

### Каноническая история

**Два подхода:**

**A) Один путь = один комикс.** Выбираем каноническую концовку → линейная история.
- Плюс: цельный нарратив, классика жанра
- Минус: 2 из 3 путей игнорируются

**B) Три параллельных трека** (как в "L'Espace D'un Soir" — французский комикс с 4 историями на разных горизонтальных уровнях).
- Плюс: сохраняет множественность, визуально уникально
- Минус: сложнее, дороже, 30 страниц тесно

**C) Хронология ДО выбора** (сцены 1–8) + три финальные страницы-сплэша с тремя исходами.
- Плюс: показывает весь нарратив и момент развилки
- Минус: финальные страницы сжаты

### Предлагаемая структура (вариант C)

| Стр. | Содержание | Стиль |
|------|-----------|-------|
| 1–2 | **Берег.** Арни, чайки, SSH-ключ. Splash-разворот, тишина | Белое на чёрном, дождь (splatter) |
| 3–5 | **Библиотека.** Пустая. Арни входит. Холст | Архитектура негативным пространством |
| 6–8 | **Barnes.** Фермер, фолк-энциклопедия. Рождение идеи | Тёплые sage-акценты |
| 9–11 | **Cogumelos.** КиберГонзо, паутинник. Верификация | Amber для опасных грибов |
| 12–14 | **Retro-Tech.** Дэн, пластинка, groove | Amber для света лампы |
| 15–17 | **Форниты.** Алиса, КиберГонзо, Дэн — команда | Три цвета — каждый персонаж свой |
| 18–22 | **Бреус.** Контракт. Зеркало. "Я был тобой" | Bordeaux, тяжёлый чёрный, лица в тени |
| 23–25 | **Развилка.** Три вертикальные полосы — три исхода | Sage / Amber / Bordeaux |
| 26–27 | **Фонарь / Мост / Кресло.** Splash-триптих | Каждый в своём цвете |
| 28 | **Команданте.** Одна фигура, белый контур на чёрном | Без цвета — чистый ч/б |

## Производство: три варианта

### Вариант 1: Профессиональный художник (ArtStation/Fiverr)

| Параметр | Значение |
|----------|----------|
| Качество | ★★★★★ |
| Бюджет | $4,000–9,000 (20–30 стр) |
| Сроки | 3–6 месяцев |
| Copyright | 100% наш |
| Процесс | Скрипт → пенсил → тушь → леттеринг |

**Где искать художников:**
- ArtStation: поиск "noir comic artist", "sin city style commission"
- Eduardo Risso (100 Bullets) — chiaroscuro мастер, доступен через галереи
- Sean Phillips (Criminal) — $675+ за пинап, полные комиксы по договорённости
- Fiverr: "$60–200/стр" диапазон, нуар-специалисты есть

**Для нуар B&W роли объединяются:**
- Пенсилер + Инкер = один человек (это норма для нуара)
- Колорист не нужен (только spot color — делает инкер)
- Леттерер: $40–50/стр отдельно

### Вариант 2: AI + художник (гибрид)

| Параметр | Значение |
|----------|----------|
| Качество | ★★★★☆ |
| Бюджет | $2,800–5,100 |
| Сроки | 2–4 месяца |
| Copyright | Частичный (AI-часть не защищена) |
| Процесс | AI-база (Midjourney/SD) → художник рефайнит → леттеринг |

**Реальность 2026:**
- Midjourney: 3–5 консистентных кадров до дрифта персонажа
- Sin City AI-модели существуют (PromptHero, Hypernatural)
- 61% digital-художников используют AI, 57% издателей — для фонов/цвета
- **Проблема:** AI-арт НЕ защищён авторским правом (дело "Zarya of the Dawn")
- **Решение:** AI = черновик, человек = финальная работа → copyright на человеческую часть

### Вариант 3: Полностью AI

| Параметр | Значение |
|----------|----------|
| Качество | ★★★☆☆ |
| Бюджет | $20–100 |
| Сроки | 1–2 недели |
| Copyright | **НЕТ** (US law) |
| Процесс | Промпт → генерация → ручная коррекция |

**НЕ рекомендуется для публикации.** Только для:
- Прототипа/питч-дека
- Внутреннего тестирования
- Визуализации сценария перед заказом художнику

## Формат и дистрибуция

### Веб (основной)

**Интерактивный web-комикс** на quest.folkup.app/comic/

| Подход | Описание | Нуар-эффект |
|--------|----------|-------------|
| **Panel-by-panel reveal** | Клик/скролл → появляется следующая панель | ★★★★★ Идеально для tension |
| **Parallax scroll** | Фон двигается медленнее переднего плана | ★★★★★ Кинематографичность |
| **GSAP-анимация** | Scroll-triggered animations (дождь, тени, свет) | ★★★★★ Sin City atmosphere |

**Технологии:** vanilla JS + GSAP (или Intersection Observer для бюджетного варианта). Без фреймворков — как весь quest.

**Open source reader:** kthoom (zero dependencies, чистый JS) или кастомный.

### Печать (дополнительный)

| Платформа | Качество | Стоимость |
|-----------|----------|-----------|
| **Lulu** (рекомендую) | ★★★★★ Мелованная бумага 80#, идеально для ч/б контраста | Бесплатная загрузка + per unit |
| IngramSpark | ★★★★☆ 40K магазинов | $49 setup |
| Amazon KDP | ★★★☆☆ Тонкая бумага | Бесплатно |

### Скачивание

- **PDF** (универсальный) + **CBZ** (Comic Book ZIP — стандарт индустрии)
- Оба за донат через Ko-fi

## Юридическая сторона

- **Стиль не защищён авторским правом** (US/EU law) — можно легально делать "в стиле Sin City"
- **Нельзя:** копировать дизайны персонажей Sin City, использовать логотип, торговые марки
- **В публичных материалах:** "classic noir comic aesthetic", "inspired by EC Comics and hardboiled tradition" — БЕЗ упоминания "Sin City style"
- **AI-арт:** не защищён copyright → гибрид или полностью ручная работа
- **Лицензия комикса:** CC BY-NC-SA 4.0 (как контент квеста) или отдельная

## Решения, которые нужны от Андрея

### 1. Подход к производству

| | Художник | AI + художник | Полностью AI |
|--|---------|--------------|-------------|
| Бюджет | $4K–9K | $2.8K–5.1K | $20–100 |
| Сроки | 3–6 мес | 2–4 мес | 1–2 нед |
| Copyright | ✅ | Частичный | ❌ |
| Качество | Максимум | Высокое | Среднее |

### 2. Структура нарратива

- **A)** Один каноничный путь → один комикс (какая концовка?)
- **B)** Три параллельных трека на странице
- **C)** Общая часть + триптих финалов (предлагаю)

### 3. Объём

- **Мини** (12–16 стр): только ключевые сцены, $2.4K–4.8K
- **Стандарт** (20–24 стр): полная дуга, $4K–7.2K
- **Расширенный** (28–32 стр): с флешбеками и деталями лора, $5.6K–9.6K

### 4. Формат дистрибуции

- Только веб (quest.folkup.app/comic/)
- Веб + PDF/CBZ скачивание
- Веб + PDF + печать (Lulu)

### 5. Следующий шаг

- **Прототип:** сгенерировать 3–5 AI-панелей для визуализации стиля (бесплатно, 1 день)
- **Поиск художника:** разместить бриф на ArtStation, посмотреть портфолио (1–2 недели)
- **Сценарий:** написать полный панельный скрипт (параллельно с поиском)
